# 📈 Stock Price Prediction with Facebook (Meta)

## 📌 Proje Hakkında  
Bu proje, zaman serisi (time series) analizi kullanarak Facebook (Meta) hisse senedi fiyatlarını tahmin etmeyi amaçlamaktadır.

## 🛠️ Teknolojiler  
Python, Pandas, NumPy, Matplotlib, Scikit-learn  

## 🎯 Amaç  
Geçmiş hisse senedi verilerine dayanarak gelecekteki fiyat hareketlerini tahmin etmek.  

## 📈 Sonuç  
Model, zaman içindeki fiyat değişimlerini analiz ederek anlamlı tahminler üretmiştir.
